HOW TO USE PHASE 1
================================================================================

STEP 1: Run Hybrid GA with Warmup
  from hybrid_ga import HybridMLGASolver

  hybrid_ga = HybridMLGASolver(
      dataset=dataset,
      surrogate=surrogate,
      feature_engineer=feature_engineer,
      warmup_fraction=0.4,  # First 40% of generations
      n_generations=100,
      mode="confidence_aware",
      ...
  )
  result = hybrid_ga.solve()

STEP 2: Extract Training Data
  from hybrid_ga import extract_training_data_from_ga

  X_ga, y_ga = extract_training_data_from_ga(result)

STEP 3: Save Dataset
  from dataset_generator import CFLPDatasetGenerator

  gen = CFLPDatasetGenerator(dataset)
  gen.save(X_ga, y_ga, "training_data_ga_derived.npz")

STEP 4: Use in Training (Phase 3)
  from training_pipeline import TrainingPipeline

  pipeline = TrainingPipeline(corpus_path="training_data_ga_derived.npz")
  pipeline.train()

================================================================================
LIMITATIONS & CONSIDERATIONS
================================================================================

CURRENT SCOPE:
  ✗ Does NOT auto-retrain (Phase 3)
  ✗ Does NOT implement competitive fitness check (Phase 2)
  ✗ Does NOT combine multiple GA runs automatically

DATA CHARACTERISTICS:
  • GA-derived samples concentrate in promising regions
  • Coverage typically 10-30% of full solution space
  • May miss extreme areas (poor and excellent solutions)
  • Not replacement for enumeration, but more efficient

CONFIGURATION:
  • Warmup must be tuned per instance
  • Requires manual training pipeline call (Phase 3)
  • Best used with confidence_aware mode (not pure_surrogate)
